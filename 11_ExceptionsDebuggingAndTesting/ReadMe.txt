Практическая работа по теме "Исключения, отладка, тестирование и логирование"
Пройдено: Возникновение исключений, отлов исключений, отладка, модульное тестирование, 
логгирование в консоли, логгирование с помощью log4j2

Задача 1. 
Цели:   Научиться использовать механизм исключений при помощи их создания и отлова в коде.
	Научиться писать конфигурацию для логирования событий при исполнении программы.
	Освоить написание тестов для классов, используя библиотеку JUnit.
Задача 2. 
	Тесты на все методы класса RouteCalculator в проекте SPBMetro.
	С помощью тестов и отладки исправлены ошибки.
